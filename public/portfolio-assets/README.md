# Portfolio Assets Structure

This folder contains all visual assets for your portfolio website.

## Folder Structure

```
portfolio-assets/
  images/
    projects/          # Project screenshots and architecture diagrams
      project1_screenshot1.png
      project1_screenshot2.png
      project1_architecture.svg
      project2_screenshot1.png
      ...
    icons/             # Technology and skill icons
      java_icon.svg
      react_icon.svg
      spring_boot_icon.svg
      docker_icon.svg
      ...
    profile/           # Professional photos and personal images
      my_photo.jpg
      profile_avatar.png
      ...
```

## Image Guidelines

### File Formats
- **PNG**: For screenshots with text or images requiring transparency
- **JPEG**: For photos and complex illustrations
- **SVG**: For icons and diagrams (scalable and lightweight)

### Optimization Tips
- Resize images to maximum display dimensions
- Compress images to reduce file size (target: 50-500KB per image)
- Use tools like TinyPNG or Compressor.io for compression

### Project Images
- High-quality screenshots of your applications
- Show main interfaces and key features
- Include architecture diagrams for backend projects
- Use consistent aspect ratios when possible

### Profile Photos
- Professional headshot with neutral background
- Good lighting and confident expression
- Square or circular format works best
- Resolution: at least 400x400px

### Technology Icons
- Use consistent icon style (flat, minimal, etc.)
- SVG format preferred for scalability
- Size: 64x64px or 128x128px
- Consider using icon packs like Font Awesome or Devicon
