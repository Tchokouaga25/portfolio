# Vidéo de fond pour la section d'accueil

## Instructions

Pour ajouter une vidéo de fond à votre portfolio :

1. **Placez votre vidéo** dans ce dossier (`/public/videos/`)
2. **Nommez le fichier** `background-video.mp4`
3. **Formats recommandés** :
   - MP4 (H.264)
   - WebM (alternative moderne)
   - Résolution : 1920x1080 ou plus
   - Durée : 10-30 secondes (boucle)
   - Taille : < 10MB pour un chargement rapide

## Exemples de vidéos gratuites

- **Pexels** : https://www.pexels.com/search/videos/technology/
- **Pixabay** : https://pixabay.com/videos/search/technology/
- **Coverr** : https://coverr.co/

## Optimisation

- Compressez la vidéo avec des outils comme HandBrake
- Utilisez des codecs modernes (H.265/HEVC si supporté)
- Testez sur mobile pour la performance

## Fallback

Si la vidéo ne se charge pas, un dégradé sera affiché à la place.
